# c3pro

## プロジェクト初期化
```
npm install
```

### プロジェクト起動
```
npm run serve
```

### 概要
App.vueに各グラフのコンポーネントが記載されています。  
コンポーネントと各ボードの紐づけは下記になっています。  

|  コンポーネント  |  ボード  |
| ---- | ---- |
|  Sales  |  今年度売上  |
|  SeasonalSales  |  季節ごとの売上変化  |
|  PurchasesCountry  |  国ごとの購入数  |
|  PriceAndSales  |  単価と売上  |
|  SoldAndSales  |  商品ごとの販売数及び売上  |
|  SalesTransition  |  商品ごとの売り上げ推移  |
